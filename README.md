## 📌 Overview

DigiTools is a fully responsive digital marketplace where users can browse and purchase premium AI tools, design assets, and productivity software. Built with a clean component architecture and modern React practices.

---

## ✨ Features

- 🛒 **Smart Cart System** — Add/remove items with real-time navbar badge count
- 🔔 **Toast Notifications** — Instant feedback on every cart action
- 💡 **Interactive Pricing Plans** — Click to activate and compare plans
- 📱 **Fully Responsive** — Pixel-perfect from 320px to 1440px+
- 🧭 **Smooth Navigation** — Scroll-aware navbar with mobile hamburger menu
- ⚡ **Blazing Fast** — Powered by Vite 8 for near-instant hot reload

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/tawchifulislam/DigiTools.git

# Navigate into the project
cd DigiTools

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
